# Student Result Management System (Python)

## Description
A simple console-based Python project to manage student results.
It allows adding student details, calculating percentage, assigning grades,
and displaying student records.

## Features
- Add student details
- Calculate total and percentage
- Automatic grade assignment
- Display all student records

## Technologies Used
- Python

## How to Run
1. Install Python
2. Clone the repository
3. Run the file:
   python student_result.py

## Future Enhancements
- Store student data in files
- Add login system
- Create GUI using Tkinter
